# PPSI 2.0 – Avaliação de Riscos de Segurança da Informação

Aplicação web para apoio à avaliação de riscos de Segurança da Informação e Privacidade, desenvolvida com base na metodologia do Programa de Privacidade e Segurança da Informação (PPSI 2.0).

## Objetivo

Disponibilizar uma interface simples para identificação, avaliação e classificação de riscos, apoiando a aplicação da metodologia do PPSI 2.0.

## Funcionalidades

- Avaliação de Impacto e Probabilidade dos riscos.
- Cálculo automático do Nível de Risco.
- Classificação qualitativa dos riscos.
- Exibição das medidas de controle recomendadas.
- Interface responsiva para utilização em navegadores modernos.

## Tecnologias

- HTML5
- CSS3
- JavaScript


## Como executar

Não é necessária instalação.

Basta abrir o arquivo `index.html` em um navegador ou acessar a versão publicada pelo GitHub Pages (https://kston.github.io/risco_ppsi_2.0/).
